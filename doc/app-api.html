To-Do List Management RESTFull API.

Request Mapping

Method URL Class Description
GET "/api/note/list" NoteController Returns list of notes for current user.
GET "/api/note/view/{noteId}" NoteController Returns note by ID.
DELETE "/api/note/update/{noteId}" NoteController Removes note.
POST "/api/note/add" NoteController Persists new note.
POST "/api/note/update/{noteId}" NoteController Updates existing note.
POST "/api/account/getUserName" com.baev.todolist.controller.rest .AccountController Retrieves current user name.
POST "/api/session/login" com.baev.todolist.controller.rest .SessionController Authenticates user in the current user session.
POST "/api/session/logout" com.baev.todolist.controller.rest .SessionController Used by the client to logout of the application.
POST "/api/session/register" com.baev.todolist.controller.rest .SessionController Creates a new user and authenticates user with this account.
GET "/api/session/isLoggedIn" com.baev.todolist.controller.rest .SessionController Checks if user is logged in.
